# Sistema de Gestão Universitária

Programa desenvolvido em java para auxiliar na gestão de uma universidade.

## Pré-Requisitos:

JDK 13 instalada no computador

## Utilização

Abrir o diretório prog2_gestao_universitaria\\src\\app e rodar pela classe App.
